PROJECT OVERVIEW
This project analyzes student performance data using SQL and Python. It demonstrates how to extract data from a database, perform analysis, and generate meaningful insights through visualization.

TECHNOLOGIES USED
PYTHON
SQL SERVER(SSMS)
PANDAS
MATPLOTLIB
pyodbc

PROJECT WORKFLOW
1.DATA STORED IN SQL SERVER DATABASE
2.PYTHON CONNECTED TO DATABASE USING pyodbc.
3.DATA FETCHED USING SQL QUERY.
4.DATA ANALYZED USING PANDAS.
5.RESULTS VISUALIZED USING MATPLOTLIB.
6.DATA EXPORTED TO CSV FOR REPORTING.

FUTURE IMPROVEMENT
USE ADVANCED VISUALIZATIONS.
HANDLE LARGER DATASET.
